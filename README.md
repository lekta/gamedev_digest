# GameDev Digest

Ежедневная сводка новостей геймдева для Unity/C#-разработчика: движок, язык, инженерные кейсы студий — коротко, по делу, на русском.

**Читать свежий выпуск: https://lekta.github.io/gamedev_digest/**

## Как это устроено

- Выпуск собирается автоматически каждый день в 10:00 МСК — GitHub Action ([`daily-digest.yml`](.github/workflows/daily-digest.yml)) запускает Claude, который работает по правилам из [`config.md`](config.md) и коммитит результат в [`digests/`](digests/).
- Страница на GitHub Pages сама подтягивает последний выпуск — после коммита ничего пересобирать не нужно.
- Пересобрать выпуск вручную: Actions → Daily GameDev Digest → Run workflow (работает и из мобильного приложения GitHub).

## Структура

- `config.md` — правила дайджеста: тематика, структура, язык, антиповторы, проверка фактов.
- `digests/` — выпуски по дням, `ГГГГ-ММ-ДД.md`.
- `memory/` — рабочая память: что освещено, очередь тем, реестр источников.
- `skill/` — скилл gamedev-digest для Claude.
- `index.html` — страница выпусков.
