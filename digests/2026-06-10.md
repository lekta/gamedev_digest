# Сводка за 2026-06-10

## Главное

### .NET 11 Preview 5 — runtime-async ускорился до неприличия
Команда .NET выкатила Preview 5 (поздно вечером 09.06 по PT) и в этот раз runtime-async получил главный апгрейд: OSR (on-stack replacement — JIT-трюк, когда долгий метод на лету переключается с «холодного» кода на оптимизированный) теперь возобновляет приостановленный async-метод сразу в оптимизированном коде, минуя общий OSR-переход. На приведённом MS-бенче с большим количеством suspension'ов время упало с 6357 мс до 457 мс — порядка x14. Сам код подвески тоже похудел (~8% на microbench), а ValueTask поверх IValueTaskSource теперь переиспользует continuation вместо аллокации. Для Unity это пока теория — CoreCLR 6.8 в движке ещё не приехал, — но именно эту картинку (async без штрафа за «корявую» приостановку) разработчики увидят, когда Unity подтянется. Если ваш код висит на горячих UniTask/await Task.Yield-петлях — отметьте: после миграции эти места перестанут болеть.
Источник: [.NET Blog](https://devblogs.microsoft.com/dotnet/dotnet-11-preview-5/)

### Nintendo Direct 09.06 — Switch 2 раздаёт индексные эксклюзивы и подкидывает Zelda Reborn
Nintendo собрала вечерний Direct ровно к концу окна Summer Game Fest и сделала ровно то, чего инди-сцена ждала: подбросила в Switch 2-эксклюзив проекты средних студий. Главный кейс с инженерной точки зрения — Orbitals от токийской Shapefarm (издатель Kepler Interactive, 03.09, Switch 2-эксклюзив): двухигроковый кооп с асимметричными ролями, нарисованные вручную катсцены Studio Massket. Это второй за неделю сигнал «маленькая студия отдаёт эксклюзив платформе ради витрины» — параллель с Solarpunk на Game Pass day-one из вчерашнего выпуска. В финале Direct Nintendo подтвердила Ocarina of Time Remake на 2026 год — слухи про переезд на движок Breath of the Wild, официально называют «rebuilt from the ground up». Switch 2 продолжает забирать инди-эксклюзивы у Steam Next Fest (15-22.06), где Unity по февральским данным держал 52,2% — посмотрим, как доля просядет в июньской волне.
Источник: [Game Informer Direct Recap](https://gameinformer.com/nintendo-direct/2026/06/09/heres-everything-announced-during-nintendos-2026-summer-direct)

### Anthropic metered billing включается через 5 дней — последний момент пересчитать экономику Claude Code
До 15.06 у пользователей Claude Pro/Max остался один уикенд, чтобы пересчитать стоимость своих агентских пайплайнов: с понедельника Agent SDK, `claude -p`, Claude Code в GitHub Actions и сторонние агенты улетают из подписочного пула в отдельный credit pool — $20 у Pro, $100 у Max 5x, $200 у Max 20x, по полной API-цене, без переноса остатка на следующий месяц. Чат в браузере/десктопе, Claude Code в терминале и Claude Cowork остаются внутри обычных лимитов подписки. Для Unity-репы на 10-30k C#-файлов с агентскими сценариями (тесты, рефакторы) разрыв между «было» и «станет» по независимым оценкам — x12-x175 по эффективной цене в зависимости от воркфлоу. Anthropic пришлёт письмо с claim-ссылкой; решение, включать ли «usage credits» (overflow по API-цене вместо отказа), нужно принять до 15.06, иначе бот молча упрётся в лимит. Точка сравнения с Polaris в Copilot (с августа) и Cursor Pro смещается на ближайшую неделю.
Источник: [InfoWorld](https://www.infoworld.com/article/4171274/anthropic-puts-claude-agents-on-a-meter-across-its-subscriptions)
