## Главное

### Rec Room закрывается сегодня — 10 лет VR-социалки и 150M игроков в анамнезе
Сегодня в полдень по тихоокеанскому времени серверы Rec Room окончательно гасят: закрывается сама игра, сайт rec.net и Rec Room Studio (тулинг под UGC). Платформа стартовала в 2016-м, прошла через все ключевые VR-устройства (Quest, PSVR/PSVR2, Pico) плюс PC/консоли без шлема, в пике собирала 150M уникальных пользователей и привлекала $300M инвестиций при оценке $3.5B (valuation — то, во сколько компанию оценили инвесторы) в раунде 2021 года. Студия прямо признала: «расходы всегда превышали выручку», а с просадкой VR-рынка дойти до устойчивой прибыли так и не получилось. После Survios и серии Meta-сокращений это второй большой VR-сигнал за месяц: модель «бесплатный массовый UGC-софт + микротранзакции + кросс-платформа» не вытащила даже флагмана, в которого вкладывались на отметках единорога. Для Unity-команд, которые делают VR/XR — это аргумент в спорах о том, оправдан ли мультиплатформенный билд: Rec Room прямо в прощальном посте отмечают, что кросс-платформа стоила им непропорционально дорого по сравнению с пользой.
Источник: [Game Developer](https://www.gamedeveloper.com/business/rec-room-shutting-down-after-failing-to-become-sustainability-profitable-)

### Microsoft Build стартует завтра — Windows Agent Runtime и Windows Agent Store с 85/15
К завтрашнему кейноуту Сатьи Наделлы (02.06, 9:30 PT, Fort Mason SF) утекли две инженерно важные новинки сверх того, что мы разбирали в выпуске за 31.05. Первая — Windows Agent Runtime: нативные API для AI-агентов прямо в shell-уровне ОС, агенты становятся первоклассной сущностью вроде процессов, а не плагином IDE. Вторая — Windows Agent Store: магазин расширений и агентов с revenue share 85/15 для разработчика (revenue share — доля выручки, которую разработчик оставляет себе после комиссии магазина), что заметно щедрее Apple App Store с его 70/30 или 85/15 после первого года. Adobe и Zoom уже подписаны как early partners. Параллельно Microsoft переписывает позиционирование GitHub Copilot из «pair programmer» в «peer programmer» — то есть инструмент, которому можно поручить таски, баги и фичи целиком, а не только подсказки. Под Unity-разработчиков напрямую ничего не анонсировали, но Windows Agent Runtime потенциально открывает путь для агентов, которые могут реагировать на состояние Editor через ОС-уровень — конкуренция MCP-серверу JetBrains и Cursor-агентам становится реальнее. Кейноут стримится на build.microsoft.com.
Источник: [Tom's Guide](https://www.tomsguide.com/computing/microsoft-build-2026-preview)
