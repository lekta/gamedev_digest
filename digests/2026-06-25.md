# Сводка за 2026-06-25

## Главное

### NVIDIA выкатила Game Agent SDK для локальных AI-NPC — но плагины только под UE5
На Unreal Fest (17.06) NVIDIA открыла бету ACE Game Agent SDK и параллельно — публичную бету PUBG Ally (17-30.06): AI-напарник Ella работает на карте Sanhok, локально на RTX с 8 ГБ VRAM, на базе Mistral-Nemo-Minitron-8B. Для Unity-программистов это критический сдвиг: впервые появился готовый кросс-движковый стек для on-device AI-агентов (распознавание речи, маленькая языковая модель, синтез голоса) с боевым кейсом — и первоклассные плагины он получил только для UE5. Unity AI / Inference Engine (бывший Sentis) пока работает в другой нише — ONNX-инференс пользовательских моделей под Unity-задачи, без коробочного «вот тебе разговорный NPC». Если в твоём проекте на горизонте AI-напарник или живые NPC, прямо сейчас Unity-команде придётся либо самостоятельно склеивать ASR + SLM + TTS поверх Inference Engine, либо ждать, пока NVIDIA доберётся до Unity-биндингов. PUBG Ally удобно как референсный профиль нагрузки: модели уровня 8B локально на консольно-классном GPU работают, но только на железе Steam Machine-уровня и выше — это сразу режет таргет на нижний край ПК-парка.
Источник: [NVIDIA Developer Blog](https://developer.nvidia.com/blog/build-on-device-ai-companions-with-the-nvidia-ace-game-agent-sdk-and-unreal-engine-5-plugins/)

### Бывшие сотрудники Xbox предупреждают: грядущие сокращения используются для сведения счётов
Game Developer 24.06 опубликовал материал с четырьмя бывшими работниками студий Microsoft, описывающими повторяющийся паттерн: подал жалобу в HR на абьюз руководителя — попал в ближайшую волну сокращений. Один из них был свидетелем во внутреннем расследовании против исполнительного директора студии, и тот, по словам сотрудника, потом «помогал составлять список на увольнение». Бывший арт-директор Halo Studios Glenn Israel в публичном посте назвал происходящее «маскировкой ретализационных действий тонкой вуалью бизнес-оправдания». На фоне ожидаемой 01.07 расширенной волны сокращений (Sony, EA, BioWare; в группе риска Bethesda, id Software, Arkane Lyon, Bungie) — это прикладной сигнал для всех, кто работает в публично-холдинговой структуре: переписку с менеджментом и HR-каналом стоит дублировать в личную почту, корпоративная история о тебе сейчас не на твоей стороне. Для Unity- и C#-инженеров на контракте с MS/Sony это особенно значимо: технические компетенции от увольнения не защищают.
Источник: [Game Developer](https://www.gamedeveloper.com/business/former-xbox-employees-warn-of-potential-for-retaliation-in-rumored-impending-layoffs)

## По диагонали

- **NVIDIA DLSS — 24.06 ещё четыре игры в списке** — Subnautica 2 получила DLSS Frame Generation и обновлённую модель Super Resolution, Mortal Shell II едет с Multi Frame Gen и ray-traced GI/reflections, The Sinking City 2 и Empulse — тоже Multi Frame Gen + RT-эффекты. По частоте обновлений списка видно, что DLSS 4 Multi Frame Gen быстро становится дефолтом для UE5-AAA; URP/HDRP-альтернативы (FSR 4 / DLSS plugin) пока подтягиваются. [Tech Critter](https://www.tech-critter.com/nvidia-dlss-news-jun-24-2026/)
- **Cultic выходит на PS5/Xbox/Switch 2 23.07** — Unity-инди-FPS одного человека (Jason Smith) спустя 4 года после PC-релиза доедет до консолей, оба эпизода в одном пакете за $19,98, издатель Atari. Кейс «Unity на ретро-FPS-боумерах, шёл прицельно от Game Maker 2D к C# в 2017» и кандидат в постмортем-копилку. Switch 2 в списке отдельно интересен: на оригинальном Switch порт сорвался из-за Tegra X1 и Unity-физики. [TechTimes](https://www.techtimes.com/articles/318935/20260623/cultic-heads-ps5-xbox-series-switch-2-july-23-both-chapters.htm)
- **Capcom Spotlight 25.06 — RE Engine как ставка на полугодие** — 30-минутный показ сегодня в 22:00 BST: Monster Hunter Stories 3: Twisted Reflection (Switch 2/PS5/PC, релиз 13.03.26 — пост-релизный роадмап), Onimusha: Way of the Sword (сентябрь), Dragon's Dogma 2: Dark Arisen (DLC 09.10). Это редкий слот, где видно концентрированную карту RE Engine; полезно для бенчмарка «куда уехали AAA-флагманы», когда Unity в этом ряду пусто. [Gematsu (RPG Site сводка)](https://www.rpgsite.net/news/20756-capcom-spotlight-showcase-june-2026-dragons-dogma-2-dark-arisen-onimusha-new-monster-hunter-stories-3)
- **Visual Studio 2026 18.7 (релиз 09.06, фиксы 18.7.1 — 16.06)** — пачка AI-фич приехала вместе: long-distance Next Edit Suggestions (Copilot видит изменения по всему файлу, не только у курсора), MCP trust validation (повторное подтверждение при изменении конфигурации MCP-серверов), индикатор контекстного окна с кнопкой «summarize», окно учёта токенов (Copilot перешёл на usage-based billing). В 18.6 — отдельная панель управления Agent Skills прямо в чате. Для Unity-репы с большими asmdef-графами long-distance NES — точка интереса: насколько хорошо Copilot будет ловить связные правки между сборками. [Visual Studio 2026 Release Notes](https://learn.microsoft.com/en-us/visualstudio/releases/2026/release-notes)
