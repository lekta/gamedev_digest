# Сводка за 2026-06-04

## Главное

### Build day 2 (03.06) — Copilot Modernization для .NET: миграция Web Forms → Blazor и Aspire-инъекция в legacy
Microsoft закрыла Build анонсом Copilot Modernization — интегрированного агента в Visual Studio 2026, который сам тянет старые приложения на свежий стек: Web Forms на Blazor, классические ASP.NET — в .NET 11, плюс ставит Aspire (фреймворк MS для облачной observability и оркестрации — собирает логи, метрики, трассировку из коробки) поверх существующих проектов. Параллельно на сессии BRK207 «Agents That Debug, Profile, and Test» Microsoft показала, как Copilot-агенты в VS 2026 теперь сами гоняют профайлер, ставят точки останова и пишут юнит-тесты по результату — целевая аудитория явно enterprise-C#. Для Unity-разработчика прямого применения нет, но это сигнал: MS методично закатывает «полу-автономного агента» в каждый шаг жизненного цикла .NET-кода, и через пару кварталов та же логика прилетит в Roslyn-аналайзеры и Rider через ACP. На фоне вчерашнего multi-agent VS Code extension вырисовывается единая платформа: одни и те же скиллы (SKILL.md), один и тот же оркестратор, разные surface-приложения.
Источник: [Visual Studio Blog](https://devblogs.microsoft.com/visualstudio/whats-coming-next-in-visual-studio-our-microsoft-build-2026-announcements/)

### Лето шоукейсов выстроилось: SGF 05.06 → Xbox 07.06 → WWDC 08-12 → Steam Next Fest 15-22 → Unreal Fest 16-19
Следующие две недели — самый плотный гейм-индустриальный отрезок года, и для Unity-разработчика особенно важна последняя точка: Steam Next Fest 15-22 июня. По данным разбора февральского Next Fest, Unity держит 52,2% демо, Unreal — 17,9%, Godot — 9%. То есть в нише «инди показывает играбельную версию», где живёт большинство C#-команд, Unity по-прежнему доминирует с явным отрывом, никаких признаков массовой миграции на UE5 после Subnautica 2 в инди-сегменте нет. Параллельно Summer Game Fest (Геофф Кейли, 5 июня, Dolby Theatre) и Xbox Games Showcase + Gears of War: E-Day Direct (7 июня) задают AAA-фон, а WWDC 8-12 — окно для анонсов visionOS 27 и нового тулинга для Apple Silicon (визионарии под Vision Pro и Mac уже начали смотреть Unity AI Inference Engine как способ гонять модели на NPU). Хороший момент пересобрать свой backlog промоматериалов под Next Fest, если планируется демо.
Источник: [Steam Next Fest](https://store.steampowered.com/sale/nextfest), [Ziva — разбор Next Fest](https://ziva.sh/blogs/steam-next-fest-2026)

### Apple Design Awards 2026: Cyberpunk 2077 Ultimate Edition забрал Visuals/Graphics, инди унесли остальное
За неделю до WWDC Apple объявила победителей Design Awards 2026. Главный технический сюрприз — Cyberpunk 2077: Ultimate Edition в номинации Visuals and Graphics: CDPR подтвердил, что REDengine 4 на Apple Silicon (M3/M4) и Vision Pro тянет полный набор технологий, включая ray tracing через Metal, — для Apple это валидация Game Porting Toolkit, для CDPR — ещё один рыночный фронт. В остальных номинациях вновь доминируют инди: Blue Prince (Innovation), Sago Mini Jinja's Garden (Interaction), Pine Hearts (Inclusivity), Consume Me (Social Impact — полу-автобиографический подростковый RPG в lo-fi). Из практических выводов для Unity-разработчика: финалистом в Visuals значится SILT (атмосферный подводный пазл, монохром, possession-механика) — Unity-проект на Apple Silicon, который через Metal-бэкенд получил награду наравне с AAA от CDPR. Кейс показывает, что упаковка под Vision Pro / Mac стала окупаемой даже для маленьких инди, и WWDC 08.06 с анонсом visionOS 27, скорее всего, добавит ещё один стимул.
Источник: [Apple Newsroom](https://www.apple.com/newsroom/2026/06/apple-reveals-winners-of-the-2026-apple-design-awards/), [AppleInsider](https://appleinsider.com/articles/26/06/02/2026-apple-design-award-winners-include-cyberpunk-2077-blue-prince)
