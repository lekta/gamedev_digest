## Главное

### Anthropic разделил подписки Claude на два кошелька — agent SDK получает фиксированный $20-$200 кредитный пул с 15 июня
13 мая Anthropic объявил, что с 15 июня programmatic-использование Claude (Claude Agent SDK, `claude -p`, Claude Code GitHub Actions и любые third-party агенты — Junie через ACP, Zed, OpenClaw и т.д.) будет тянуться из отдельного фиксированного пула: $20 для Pro, $100 для Max 5×, $200 для Max 20×. Кредиты не переносятся на следующий месяц, после исчерпания работа возможна только по pay-as-you-go API-расценкам. Параллельно снят апрельский запрет на третьесторонние агенты через подписку — но теперь это «свобода с потолком». Для Rider-разработчика, гоняющего Junie через ACP, это означает простую вещь: $20 Pro перестаёт окупаться, как только Junie начинает выполнять что-то сложнее автодополнения. По индустриальным замерам, эффективная цена programmatic-использования вырастает в 12-175× в зависимости от воркфлоу — то есть Anthropic фактически закрывает лазейку, через которую дешёвые подписки субсидировали API-нагрузку под видом разговора с человеком.
Источник: [The New Stack](https://thenewstack.io/anthropic-agent-sdk-credits/) / [The Register](https://www.theregister.com/ai-ml/2026/05/14/anthropic-tosses-agents-into-the-api-billing-pool/5240748)

### Katsuhiro Harada возглавил VS Studio — новую японскую дочку SNK под action/PvP
1 мая в Токио зарегистрирована VS Studio SNK Co., Ltd., 12 мая объявлено публично: 31-летний ветеран Bandai Namco и идеолог Tekken (ушёл из Bandai в декабре 2025) теперь Representative Director и CEO. Студия будет делать competitive PvP / action-проекты — Харада не исключил работу над файтингами SNK, но напрямую этого пока не подтверждает. На фоне волны закрытий (Survios, Spiders, Visual Concepts) это уже второй за май кейс открытия новой студии после Drecom Creators Studio (14 мая) — оба из Японии, и оба под крылом крупного издателя. Технологический стек VS Studio пока не раскрыт; для Unity-программиста это сигнал, что японские издатели всё чаще запускают новые подразделения под ветеранов вместо найма по аутсорс-контрактам — а значит, спрос на C#-разработчиков в Японии может вырасти, если студия не пойдёт сразу в UE5.
Источник: [Game Developer](https://www.gamedeveloper.com/business/tekken-veteran-katsuhiro-harada-to-lead-new-snk-subsidiary) / [SNK Press Release](https://www.snk-corp.co.jp/us/press/2026/announcement-of-new-studio-establishment-katsuhiro-harada-appointed-as-representative/)

### VS Code 1.120 — Agents window переехал в Stable, BYOK теперь показывает расход токенов, plan-mode редактируется inline
Microsoft 13 мая выкатил очередной weekly-релиз с упором на агентскую работу: Agents window (отдельный воркспейс для AI-сессий и кастомизаций) вышел из preview в Stable, plan mode при работе с Claude/Copilot CLI получил inline-control для правки плана прямо в чате без перехода в отдельный таб. BYOK (Bring Your Own Key — режим, когда подключаешь свой API-токен вместо подписки Copilot) теперь показывает реальный расход токенов прямо в интерфейсе — критичная вещь именно сейчас, учитывая параллельный sweep Anthropic по metering (см. выше). Plugin-инсталляция через Copilot CLI автоматически подхватывается VS Code — один `copilot plugin install` покрывает обе поверхности. Для Rider-разработчика это в основном «удалённая» новость, но релиз закрывает один из последних UX-аргументов в пользу VS Code в C#-нише: dedикейтед-окно для AI-агентов с per-session-историей. Вопрос, ответит ли JetBrains аналогом в Rider 2026.2 (EAP 2 ожидается на неделе).
Источник: [VS Code 1.120 release notes](https://code.visualstudio.com/updates/v1_120) / [Visual Studio Magazine](https://visualstudiomagazine.com/articles/2026/05/13/hands-on-with-the-new-agents-window-in-vs-code-1,-d-,120.aspx)
