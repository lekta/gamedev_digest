### 2K режет 31st Union — Project Ethos переделывают в PvP-roguelike
8 мая 2K сократили часть штата 31st Union (Take-Two, Сан-Матео) и одновременно сменили жанр Project Ethos: из extraction-шутера он становится «skill-based PvP roguelike». Глава студии Бен Бринкман: «изменения нужны», но он «никогда не был более уверен» в проекте. Кейс рифмуется с Marvel Snap → Godot и MindsEye из covered: AAA-боссы продолжают режут команды и пересобирают игры через два-три года после анонса, чёткой стратегии не видно.
Источник: [Game Developer](https://www.gamedeveloper.com/business/2k-lays-off-staff-at-31st-union-working-on-project-ethos)

### Unity Q1-call: Unity AI взяли 70% за пять дней + Commerce Platform с SciPlay и Voodoo во II квартале
На звонке 7 мая помимо самих финансов Unity раскрыли две детали. Первое: open beta Unity AI с 4 мая получила 70% attach за пять дней и 90% opt-in в data framework, который кормит Vector. Второе: Unity Commerce Platform — единый дашборд для каталогов, цен и пайаутов на mobile/web/PC — выходит в Q2 2026 с SciPlay (соц-казино) и Voodoo (гипер-кэжуал) как лонч-партнёрами. Это попытка увести разработчиков из IAP-стека Apple/Google в собственную монетизацию; если зайдёт, рано или поздно появится в Hub как «native» вариант.
Источник: [Motley Fool — Q1 2026 earnings call transcript](https://www.fool.com/earnings/call-transcripts/2026/05/07/unity-u-q1-2026-earnings-call-transcript/)

### Anthropic удвоил лимиты Claude Code и снял пиковые ограничения для Pro/Max
6 мая Anthropic удвоил пятичасовые лимиты Claude Code для Pro, Max, Team и Enterprise и убрал ограничения по часам пик; на API tier 1 — +1500% input tokens/min, +900% output. Причина — деал с SpaceX о подключении дата-центра Colossus 1 (Мемфис, ~220k Nvidia GPU, 300 МВт). Параллельно Anthropic заявил $30B годовой выручки и сослался на корпоративных клиентов (Uber, Netflix), которые гонят Claude Code в проде. Для C#-практиков это первый случай, когда обычный Pro-тариф реально вытягивает большие сессии на крупных Unity-репах без дробления контекста.
Источник: [9to5Google](https://9to5google.com/2026/05/06/claude-code-is-getting-higher-usage-limits-doubled-for-most-users/)
